> 安装淘宝npm镜像

```
npm install -g cnpm --registry=https://registry.npm.taobao.org
```

> 全局安装vue命令行工具
```
cnpm install -g vue-cli
```
> cmd创建文件夹

```
md 文件夹名
```

> 初始化
> 

```
vue init webpack my-first-vue-project
```

> 启动

```
npm run dev
```
